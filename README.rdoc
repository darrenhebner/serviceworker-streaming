{== WIP

The is a toy rails app used for experimenting with streaming inside of a serviceworker.
The idea is that the service worker is able to return the "shell" instantly from cache while still fetching the requested page. This is different from the typical PWA app shell structure.

Learn more here: https://jakearchibald.com/2016/streams-ftw/#creating-one-stream-from-multiple-sources-to-supercharge-page-render-times
